# System Scope

## Objectif
Monitoring centralisé durci pour Airstream 2004. Le produit doit remplacer le panneau actuel qui lit les niveaux des réservoirs d'eau fraiche, grise et noire, et fourni un interrupteur pour la pompe à eau.
Le système fera de l'aquisition multi-protocoles (Analogique, I2C, BLE) et visualisation temps réel.

## Fonctionnalités
- Mesure le niveau du réservoir d'eau fraîche
- Mesure le niveau du réservoir de grise
- Mesure le niveau du réservoir de noire
- Affiche le niveau de jusqu'à 3 réservoirs de propane
- Affiche l'état de jusqu'à 4 batteries
- Affiche l'état de jusqu'à 3 chargeurs
